Cài đặt API Key

Vui lòng nhập API Key của bạn từ Google AI Studio để sử dụng tính năng phân tích bằng AI.

Cách lấy API Key:

  1. Truy cập trang Google AI Studio và đăng nhập bằng tài khoản Google.
  2. Nhấp vào nút "Get API key".
  3. Chọn "Create API key in new project".
  4. Sao chép chuỗi API Key hiện ra và dán vào ô ở trên.
